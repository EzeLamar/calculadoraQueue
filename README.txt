Programa ejemplo de uso de colas IPC entre 2 programas (cliente, servidor)
	+El server inicializa la cola para la comunicacion
	+El Cliente solicita una de las 5 operaciones (suma, resta, mult, div, DTB y BTH)
	+El Server realiza el calculo y le contesta al Cliente con el resultado
	+El Cliente muestra por pantalla el resultado recibido.
